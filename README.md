# Electronixa_Gestion_Produits
Ce projet consiste à développer une application de bureau pour la gestion du catalogue de produits d'Electronixa, une PME spécialisée dans la distribution de produits électroniques et informatiques. L'objectif est de remplacer la gestion actuelle basée sur des fichiers Excel par une solution moderne, sécurisée et centralisée.
